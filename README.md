# JuniorDevs
VDI's Youth Outreach Junior STEM Developer Network

This repository is an exemplar software/hardware/firmware & engineering example project to help young devs learn the networking ropes that go along with science. 

All new STEM workers get interaction through social networks just like at home. In fact, it was science and engineering workers that built those networks you use. 

All software needs revisions and collaboration. 

You show you can prosper in a distributed dev network by learning how they function.

If you don't know how to do something then try go look it up IN Wiki! 
If you don't know how to do something and can't find in Wiki, go look it up and POST IT IN Wiki!
See? You're helping already!
